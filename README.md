# Programmes requis 

## Installer make 

```
 apt-get install build-essential make
```

## Installer SDL

```
sudo apt-get install libsdl1.2-dbg
```

[Guide d'installation](https://wiki.libsdl.org/Installation)


# Utilisation 

```
# Générer l'exécutable 
make 

# Lancer l'exécutable
./exec 
```
